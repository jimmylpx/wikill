Wi-Kill User Guide

Download

Download aplikasi Wi-Kill:
https://bit.ly/36rWgTM

Setup

1. Nyalakan wi-kill dengan menggunakan micro usb yang terhubung dengan charger atau daya dari port usb apapun.
2. Install aplikasi yang bisa anda dapatkan melalui link download diatas.
3. Setelah itu buka pengaturan wifi handphone anda dan cari SSID bernama Wi-Kill dan konekan hp anda ke SSID tersebut.
4. Biasanya SSID Wi-Kill tidak meminta password wifi, namun jika meminta silahkan masukkan "rootaccess" tanpa tanda petik.
5. Jika sudah konek, silahkan buka aplikasi Wi-Kill, kemudian anda akan disuruh untuk merubah SSID dan password wifi dari Wi-Kill.
6. Setelah berhasil, silahkan ulangi langkah nomor 3 tapi hubungkan hp ke SSID wi-kill yang sudah anda ganti beserta passwordnya lalu silahkan kembali ke aplikasi wi-kill.

Pemakaian

Pemakaian wi-kill bisa dengan 3 cara, yaitu:

1. Langsung menekan tombol One Push For All sebanyak 2x sampai menyala biru. Fitur ini dapat mengaktifkan wifi deauther ke semua jaringan tersedia hingga 30 meter. Fitur ini hampir mirip dengan 2.4Ghz Jammer. Harap gunakan dengan bijak.
2. Melalui web browser. Setelah konek ke SSID wi-kill, silahkan akses ip 192.168.4.1 lewat browser kalian untuk mengakses interface wi-kill.
3. Melalui aplikasi wi-kill. Untuk penggunaannya silahkan lihat proses SETUP diatas.

Untuk penggunaan fitur wi-kill silahkan pilih ssid target dengan cara cek box lalu tekan tombol serang. Pada menu penyerangan silahkan lalukan serangan apa yang diinginkan.
